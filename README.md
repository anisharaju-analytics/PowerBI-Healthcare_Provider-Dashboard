# PowerBI-Healthcare_Provider-Dashboard
Built an interactive Healthcare Provider Dashboard in Power BI to analyze billing, treatment costs, insurance coverage, and patient service utilization. Features KPI tracking, geographic analysis, departmental performance, procedure insights, and dynamic visualizations to support healthcare operational and financial decision-making.
## 📊 Dashboard Features
Key Performance Indicators (KPIs)
- Total Billing Amount
- Total Treatment Cost
- Total Medication Cost
- Total Insurance Coverage
- Total Out-of-Pocket Expenses
- Total Room Charges
- Average Cost Metrics
## Analytics & Insights
- Billing Amount by City
- Billing Amount by Procedure
- Department-wise Billing Analysis
- Diagnosis and Service Type Analysis
- Geographic Patient Distribution
- City/State Location Analysis
## Interactive Features
- Dynamic Filters and Slicers
- Light & Dark Mode Toggle
- Interactive Maps
- Responsive Dashboard Navigation
- Drill-Down Analysis
## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Bookmarks & Buttons
- Interactive Maps
- Data Visualization
## 📈 Business Value
This dashboard helps healthcare organizations:
- Monitor revenue and healthcare cost distribution.
- Analyze insurance coverage and patient out-of-pocket expenses.
- Identify high-performing departments and procedures.
- Evaluate patient service utilization patterns.
- Support operational and financial decision-making through actionable insights.
